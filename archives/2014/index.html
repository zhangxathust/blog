如何使用Hexo和GitHub来新建博客

使用github搭建博客有很多好处：

1.简洁，不用自己管理数据库，申请域名等
2.支持HTML、markdown语法
3.只需要管理博客内容

基于node的Hexo搭建github博客步骤

1.安装git
1).mac下安装

2).Linux下安装（推荐软件源安装方式，省事省时；当然如果可以也可以源码安装，目录统一，方便卸载）
RedHat、CentOS下安装：yum install git(貌似是这个命令)
Ubuntu等的安装：apt-get install softname1 softname2(貌似是这个命令)

3).windows下安装
search ‘msysgit’ and install

安装好git之后，配置github账户

1).申请github账户，免费账户只能create public repository，数量不限；交钱的账户可以建立private repository.

2).使自己的电脑能够通过git协议来pull\push自己的repository，
即配置SSH keys，这个keys通过命令行产生（ssh-keygen -t rsa -C “XXX@company.com”）
生成的ssh keys存放在用户的根目录下的.ssh文件夹下(这个文件夹是隐藏的)
mac用户和Linux用户输入一下命令

windons用户通过explorer查看c:/users/administrator/

添加ssh keys到personal setting，即将id_rsa.pub中的内容
