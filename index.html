“宽带维修——联通公司”

如果遇到联通宽带配的猫，中间的一个灯闪烁，则可能是猫绑定的宽带账号过期了，现将设置教程记录如下：

  1. 登陆进入无线路由器，一般的地址为192.168.1.1，进入之后选择“高级设置”=》“LAN口设置”，修改ip地址为192.128.1.2或者3…

  2. 拔掉无线路由器网线WAN口进入四个LAN口中的一个

  3. 浏览器中输入ip(192.168.1.1)，进入猫管理界面，中兴公司的猫账户密码是：admin

  4. 在猫管理界面中选择“网络”=》“PON信息设置”，输入LOID信息宽带信息+四位数(具体询问宽带维修人员)，点击确定即可

  5. 将路由器LAN口网线拔掉，插入WAN口中，浏览器中输入ip(192.168.1.2)，进入输入宽带账户和密码，(按照设置向导设置即可)

  6. 重启路由器

如何使用Hexo和GitHub来新建博客

使用github搭建博客有很多好处：

1.简洁，不用自己管理数据库，申请域名等
2.支持HTML、markdown语法
3.只需要管理博客内容

基于node的Hexo搭建github博客步骤

1.安装git
1).mac下安装

2).Linux下安装（推荐软件源安装方式，省事省时；当然如果可以也可以源码安装，目录统一，方便卸载）
RedHat、CentOS下安装：yum install git(貌似是这个命令)
Ubuntu等的安装：apt-get install softname1 softname2(貌似是这个命令)

3).windows下安装
search ‘msysgit’ and install

安装好git之后，配置github账户

1).申请github账户，免费账户只能create public repository，数量不限；交钱的账户可以建立private repository.

2).使自己的电脑能够通过git协议来pull\push自己的repository，
即配置SSH keys，这个keys通过命令行产生（ssh-keygen -t rsa -C “XXX@company.com”）
生成的ssh keys存放在用户的根目录下的.ssh文件夹下(这个文件夹是隐藏的)

windons用户通过explorer查看c:/users/administrator/

添加ssh keys到personal setting，即将id_rsa.pub中的内容

2) 下载binary文件，解压缩，环境变量设置(自行google mac如何设置环境变量)这样可以对node的所有相关文件进行管理；

想拥有一个独特的自己的域名吗？（发动思维区去想吧，说不定哪天就被哪位大卡给收购了！！！）

如果想拥有一个像www.baidu.com这样的访问链接，就需要用户去注册一个自己喜欢的域名。

1).注册域名，dnspod上提供注册域名：https://domains.dnspod.cn/

2).域名解析，dnspod提供免费的域名解析

3).将自己的github博客指向这个域名，第一修改_config.yml中的url属性到自己的域名上，其次在source目录下添加CNAME文件，内容为自己注册的域名.
